step 1: Create virtual enviornment

        b)cd myenv
        c)cd Scripts
        d)for activation
             ./activate
        e)back to main folder
          cd ../..

step 2: django install
         python -m pip install django

step 3: Project Creation
       a)django-admin startproject firstproject(Project name)
       b)cd firstproject
       c)python manage.py runserver(for running the project)

step 4: Create application
        a)django-admin startapp firstapp(application name)
        b)python manage.py runserver(for running the application)
